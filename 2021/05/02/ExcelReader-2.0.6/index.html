ExcelReader 2.0.6 version now released！

0x01 新功能

ExcelReader 2.0.6 版本重磅更新了。增加了对多个sheet标签页的支持。

0x02 安装

在IDE里面进行安装

打开 “File” - “Settings” - “Plugins”，选择 “Marketplace”，输入 ExcelReader ，点击 install 即可安装到IDE。

不同版本或者平台的IDE，上述的打开方式可能略有差别

通过web安装
