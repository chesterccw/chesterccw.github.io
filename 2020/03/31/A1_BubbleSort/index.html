冒泡排序

描述

“冒泡”排序是数列排序的算法之一。

规则

有长度为n的数列。设 index = n， 比较第 n 位和第 n-1 位，如果右边的数字较小，则交换其位置。比较完成后，逐一向左移动 index，同样比较数字。如果右边的数字较小，则交换其位置。重复同样的操作，直到 index 移动到最左边。

在上面的操作中，数列中最小的数字已经移动到了最左端，然后使最左端的数字排序。

之后将 index 移动到最右端，重复相同的操作，直到所有的数字都被排序。这样就完成了整个过程。
