IDEA Plugin - ExcelDiffer

2020-09-13

0x01 介绍

ExcelDiffer 是一个对比Excel类格式文件的IDE插件，支持 .xls, .xlsx, .csv 三种格式。

0x02 安装

在IDE里面进行安装

打开 “File” - “Settings” - “Plugins”，选择 “Marketplace”，输入 ExcelDiffer ，点击 install 即可安装到IDE。

不同版本或者平台的IDE，上述的打开方式可能略有差别

通过web安装

点击前往ExcelDiffer插件主页

0x03 使用

鼠标点击选中一个后缀是 .xls 或 .xlsx 或 .csv 的文件，然后右击选择 Add to ExcelDiffer -> Left。然后选择另一个文件，右击选择 Add to ExcelDiffer -> Right 即可自动加载 文件内容，并显示比较结果。

右上角显示的是比较结果（有多少个不相同的地方）。

左上角是操作区域，可以在不同的结果之间跳转。

  • 跳转到上一个不同之处
  • 跳转到下一个不同之处
  • 刷新，从磁盘重新加载当前的文件

界面的最下方显示左右两边选中的同一行的纵向对比。
